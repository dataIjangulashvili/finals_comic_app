# Manga tracker App

მარტივი აპი მიმდინარე მანგების კლასიფიკაციის, საკუთარი სიის აწყობის და ახალი update-ების ინფორმაციის გასაგებად.

## Features
home ზე გამოტანილი გვაქვს Jikan API ს გამოყენებით MyAnimeList დან წამოღებული მანგების ინფორმაცია.
.
მოძებნის ფუნქცია ეძებს იმავე API-ში სახელების გამოყენებით.
.
შესაძლებელია რამოდენიმე მანგის follow და შემდგომ წაკითხული chapter-ების აღრიცხვა და შენახვა.
.
My List-ში შენახული ინფორამცია ჩანს home page-ზეც.
.
დამატებითი ფუნქციები:
.
ექსპორტის გამოყენებით შეგვიძლია სიის ამოღება csv ფორმატით აპლიკაციიდან რათა მოხდეს საჭიროების შემთხვევაში ამ სიის backup და გაზიარება.
